Como o Algoritmo Verifica LD/LI?

Para determinar se um conjunto de vetores é Linearmente Dependente (LD) ou Linearmente Independente (LI), o algoritmo segue os seguintes passos:

  1. Coleta de Vetores: Primeiro, o algoritmo recebe os componentes (X, Y, Z) de cada vetor. Neste caso, esperamos 3 vetores 3D.
  2. Criação da Matriz: Em seguida, esses vetores são organizados em uma matriz. Cada vetor se torna uma linha (ou coluna, dependendo da convenção) da matriz. Por exemplo, para vetores $v1 = (x1, y1, z1)$, $v2 = (x2, y2, z2)$ e $v3 = (x3, y3, z3)$, a matriz será:

  4. Verificação do Resultado:
    • Se $determinante === 0$: Os vetores são Linearmente Dependentes (LD). Isso significa que pelo menos um dos vetores pode ser expresso como uma combinação linear dos outros, ou, geometricamente, os vetores estão no mesmo plano (ou linha) passando pela origem.
    • Se $determinante \neq 0$: Os vetores são Linearmente Independentes (LI). Isso indica que nenhum dos vetores pode ser escrito como uma combinação linear dos outros. Geometricamente, em um espaço 3D, três vetores LI formam um "espaço" e não estão confinados a um único plano.
  5. Visualização: Finalmente, os vetores são plotados em um gráfico 3D para uma representação visual do seu arranjo espacial.
